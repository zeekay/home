# Changelog

All notable changes to zOS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Features

- Added centralized app versioning system (`appVersions.ts`)
- Added app registry with categories and bundle IDs (`appRegistry.ts`)
- Added GitHub release automation
- Added version management scripts

### Infrastructure

- Auto-increment build numbers on deploy
- Generate changelog from git commits
- Create GitHub releases on tag push

## [4.2.0] - 2024-12-27

### Features

- **App Store**: New app catalog with versions and categories
- **Safari**: Tab browsing and bookmarks
- **Terminal**: WebContainer integration
- **Music**: Spotify and SoundCloud integration
- **Photos**: Instagram API integration
- **Calendar**: Cal.com scheduling integration

### AI Apps

- **Hanzo AI**: AI assistant powered by Claude
- **Zoo**: Decentralized AI research assistant
- **Lux Wallet**: Quantum-safe cryptocurrency wallet

### Developer Tools

- **Xcode**: Monaco Editor IDE
- **Terminal**: Full shell emulation
- **GitHub Stats**: Contribution tracking

### Improvements

- Improved window management
- New notification system
- Better keyboard shortcuts
- Dark mode support across all apps

## [4.1.0] - 2024-12-15

### Features

- Mission Control for space management
- Improved Finder with Quick Look
- New desktop widgets

### Bug Fixes

- Fixed window resize issues
- Fixed dock animation glitches

## [4.0.0] - 2024-12-01

### Breaking Changes

- Complete UI overhaul with macOS Big Sur design

- Restructured component architecture

### Features

- macOS-style desktop interface
- Interactive dock
- Menu bar with system controls
- Lock screen and boot sequence
